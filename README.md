# Real-World Retail Sales Data Analysis Project

## Retail Sales Data Analysis and Visualization Using Python

---

# 1. Introduction

This project focuses on analyzing a real-world retail sales dataset using Python libraries such as Pandas, Matplotlib, and Seaborn.

The objective of the project is to perform data analysis, identify business trends, generate visual insights, and understand customer purchasing behavior from retail sales data.

The project demonstrates practical application of data science skills in a real-world business environment.

---

# 2. Objectives

The main objectives of this project are:

* To analyze retail sales data
* To identify trends and patterns in customer purchases
* To perform statistical and visual analysis
* To generate business insights from data
* To understand relationships between sales, profit, and customer behavior
* To improve practical data analysis skills

# 4. Dataset Information

Dataset Name: Supermarket Sales Dataset

The dataset contains information related to:

* Product lines

* Sales amounts
* Gross income
* Payment methods

* City and branch information

The dataset was loaded in CSV format and analyzed using Python.

# 9. Key Insights

The following insights were observed from the analysis:

1. Certain product categories generated significantly higher sales than others.
2. Electronic payment methods and cash payments were widely used by customers.
3. Gross income showed a positive relationship with sales amount.
4. Customer purchasing behavior varied across product categories.
5. Some branches and cities generated higher overall sales performance.
